# ЯсноТекст

> Программа, которая превращает «недоступные» цифровые документы
> в удобную среду для людей с нарушениями зрения и дислексией.

Учебный проект по дисциплине «Основы проектной деятельности».

## Проблема

Миллионы цифровых документов — учебники в PDF, сканы, методички в DOCX —
недоступны для людей с особенностями зрения или дислексией. Часто это
просто картинки, которые невозможно ни увеличить без потери качества,
ни перекрасить, ни озвучить стандартными программами.

Для человека с особенностями восприятия такой файл — это запертая дверь
к знаниям. ЯсноТекст помогает её открыть.

## Решение

ЯсноТекст работает как «умный посредник» между документом и читателем:

1. Открывает PDF и DOCX, включая сканы.
2. Распознаёт текст через OCR (Tesseract).
3. Применяет один из трёх профилей доступности — и весь интерфейс
   адаптируется под потребности пользователя.

## Профили

| Профиль | Шрифт | Размер | Цветовая схема | Особенности |
|---|---|---|---|---|
| Стандартный | Segoe UI | 14pt | Чёрный на белом | Базовый режим |
| Слабовидение | Arial Bold | 28pt | Жёлтый на чёрном | Контраст 21:1, крупные элементы |
| Дислексия | OpenDyslexic | 18pt | Тёмно-коричневый на бежевом | Расширенные интервалы между буквами и словами |

Каждый профиль меняет не только текст документа, но и весь интерфейс
приложения — кнопки, меню, панели.

## Возможности

- 📂 Открытие PDF и DOCX
- 🔍 OCR для сканов (Tesseract, русский и английский)
- 👁️ Профиль «Слабовидение»: контрастные цвета, крупный шрифт
- 📖 Профиль «Дислексия»: OpenDyslexic, расширенные интервалы
- 🎨 Настройка шрифта, размера, цветов, межбуквенного интервала
- 💾 Сохранение настроек профиля между сессиями
- ⌨️ Горячие клавиши для быстрого переключения (Ctrl+1/2/3)

## Технологии

- **.NET 8** + **WPF** — платформа и UI
- **Tesseract.NET** — распознавание текста
- **PdfPig** — парсинг PDF
- **DocumentFormat.OpenXml** — парсинг DOCX
- **OpenDyslexic** — специальный шрифт для людей с дислексией

## Сборка и запуск

Требования:
- Windows 10 или 11
- .NET 8 SDK ([скачать](https://dotnet.microsoft.com/download))

```bash
git clone https://github.com/Serouke/yasnotext.git
cd yasnotext
dotnet build
dotnet run --project src/YasnoText.UI
```

Для работы OCR дополнительно нужно скачать языковые данные Tesseract
и положить их в папку `src/YasnoText.UI/tessdata/`:

- [rus.traineddata](https://github.com/tesseract-ocr/tessdata/raw/main/rus.traineddata)
- [eng.traineddata](https://github.com/tesseract-ocr/tessdata/raw/main/eng.traineddata)
